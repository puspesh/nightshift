# Platform Reference

> Platform-specific constraints and conventions for content agents.

## Twitter / X

| Setting | Value |
|---------|-------|
| Character limit | 280 |
| Thread max | 10 tweets |
| Hashtags | 0-2 per tweet |
| Links | Shortened automatically, ~23 chars |
| Images | Recommended but not required |

**Conventions:**
- Front-load the hook in the first line
- Use line breaks to create visual breathing room
- Threads: number each tweet (1/, 2/, etc.)
- Quote tweets > reply threads for engagement
- Avoid starting with "@" (limits visibility)

## LinkedIn

| Setting | Value |
|---------|-------|
| Character limit | 3,000 |
| Line breaks | Crucial for readability |
| Hashtags | 3-5 at the end |
| Links | Place in first comment (algorithm penalty in body) |
| Tone | Professional but conversational |

**Conventions:**
- First 2 lines must hook (they show before "see more")
- Use short paragraphs (1-2 sentences each)
- Personal stories outperform generic advice
- End with a question or clear CTA
- Avoid excessive emoji (1-2 max per post)

## Cross-Platform Rules

- Never cross-post identical content — adapt per platform
- Each platform version should feel native, not copy-pasted
- Respect character limits strictly — don't truncate, rewrite
- When in doubt about tone, check `knowledge/style-guide.md`
